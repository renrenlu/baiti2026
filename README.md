# 百题大过关·小升初语文互动练习

基于用户提供的《百题大过关·小升初语文 基础百题（修订版）》PDF 制作的离线互动 HTML。

## 使用

直接双击 `index.html`，或在本目录启动任意静态文件服务器。页面不依赖网络。

## 内容对应

- 书本正文第 1—154 页对应原 PDF 第 6—159 页，共 154 张原题扫描。
- 第一章 001—077 已转写为逐题文字卡，并与原书第 4—13 页、答案册第 1—3 页逐题对应。
- 正文连续题号 001—509 依据原书答案册页码范围定位到唯一答案页。
- 10 套“基础冲关”分别对应答案册第 24—30 页中的相关页。
- 全书保留 PDF 扫描页；第一章可在“文字题卡 / 原书扫描”之间随时切换。

## 交互功能

- 章节与书页导航、原页放大、阅读缩放。
- 第一章逐题作答、答案折叠、选择题即时核对、主观题自评、题号跳转。
- 图片加载状态与失败重试，静态资源版本号用于规避微信旧缓存。
- 输入原书题号后显示对应答案页。
- 基础冲关按套题自动显示对应答案页。
- 本页作答、收藏、完成打卡、答对/订正记录。
- 所有记录保存在浏览器 `localStorage` 中。

## 键盘快捷键

- `←` / `→`：文字题卡中切换上一题 / 下一题；扫描模式中切换上一页 / 下一页
- `A`：核对当前题答案 / 显示扫描答案
- `D`：切换本页完成状态
